Es un concepto imortante “Documento Modelo de Objeto”

- Los usuarios deben poder interactuar con los elementos en una pagina web, eso significa que el documento HTML debe representarse de forma que el codigo JavaScript pueda consultarlo y actualizarlo

Esa es la funcion del DOM

- es un modelo de objetos en el archivo HTML

Los desarrolladores web interactual con el DOM a traves de JavaScript para:

- actualizar el contenido
- configuracion de eventos
- animacion de elementos HTML

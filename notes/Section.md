Este elemento se utiliza parara definir semanticamente secciones individuales del articulo y es importante que las secciones deben haber elementos de encabezado para describir semanticamente la seccion.

```html
<article>
  <header>
    <h2>My summer vacation</h2>
    <p>
      This summer, I went to the beach with my family. We had a great time
      swimming in the ocean and building sandcastles. We also went on a boat
      tour and saw some dolphins. It was a wonderful vacation!
    </p>
  </header>
  <main>
    <p>This was my most memorable summer vacation so far!</p>
  </main>
  <section>
    <h3>Photos from the trip</h3>
    <p>Here are some photos from our summer vacation:</p>
  </section>
</article>
```

Tambien se puede utilizar elementos de seccion para describir las diferentes secciones de un sitio web, ya que el elemento de section no requeire del elemento article.

Todo depende de se desee describir semanticamente la pagina web.

```html
<body>
    <section>
        <h2>About me</h2>
        <p>My name is Jane, I'm a developer .< /p>
    </section>
    <section>
        <h2>Technologies I know</h2>
        <ul>

            <li>HTML</li>
            <li>CSS</li>

    </section>
</body>

<ul>
```

# HTML and CSS in Depth - Meta

This repository contains my exercises, labs, notes, and projects completed while taking the **HTML and CSS in Depth** course offered by **Meta** on Coursera.

## About the Course

This course focuses on advanced HTML and CSS concepts used in modern front-end development, including:

- Semantic HTML
- Metadata and SEO fundamentals
- Forms and validation
- Multimedia elements
- CSS Flexbox
- CSS Grid
- Selectors and specificity
- Pseudo-classes and pseudo-elements
- CSS transitions and transformations
- CSS animations
- Responsive layouts
- Front-end debugging
- Portfolio project development
